Traveler Journey Lab

I created the Traveler Journey Lab for a variety of reasons. The first was to provide a sandbox for designers to play in which mimicked the existing production pages within the Traveler Journey bet. The second was to make a prototype as realistic as possible to test on human beings in our user-testing labs across the world. Finally, the TJ Lab acts as living breathing documentation for the engineers to implement tested designs. This documentation leaves no questions unanswered about responsive behaviour, transition/animation timings, hover interactions etc.

Highlights

  • Mobile first, responsive prototype.
  • Built with Polymer to handle data management for the Multilist work.
  • Deployed on the Expedia network so anyone in the company can access.

When Wotif was acquired by Expedia, we were tasked with solving a unique problem. Everytime Expedia would acquire a new brand they would roll up the new brands pages into Expedia's templating solution which would present to Google near identical pages (give or take the logo and a couple of highlight colours). Google picked up on this and started to penalise brands within the same region. For example, in the US region, Expedia/Travelocity/Orbitz/Cheaptickets were all competing for a spot on Google with the same content and design being served, which Google did not appreciate. It was our job to navigate our way out of this predicament.

In true test and learn fashion we incrementally discovered that the only thing that appeased Google was radically different pages, this included design, content and even the code that built the pages.

Unfortunately time was not on our side as all the regional brands were leaking cash daily, so I was unable to create an extensive styleguide for each brand (in a perfect world this is the approach I would of taken). Instead we defined five themes which would be distibuted amongst the brands and help guide the design and content for each. For example, every module on a page that used the Reviewer theme, would find a way to reference reviews or some other form of user generated content.
